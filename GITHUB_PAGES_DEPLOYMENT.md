# SOVIRA GLOBAL GitHub Pages Deployment

Use this package for a free GitHub Pages launch.

## Recommended free setup

Create a GitHub repository named exactly:

`YOUR-GITHUB-USERNAME.github.io`

Example: if your GitHub username is `soviraglobal`, create:

`soviraglobal.github.io`

This gives you the free live URL:

`https://YOUR-GITHUB-USERNAME.github.io/`

This site package is optimized for root hosting, so the username.github.io repository is the easiest option.

## Upload steps

1. Create a new public GitHub repository named `YOUR-GITHUB-USERNAME.github.io`.
2. Extract this zip file on your computer.
3. Upload the contents inside the extracted folder to the repository root. Do not upload the zip file itself.
4. Commit the files to the `main` branch.
5. Go to repository Settings > Pages.
6. Set Source to `Deploy from a branch`.
7. Set Branch to `main` and folder to `/root`.
8. Save.
9. Open `https://YOUR-GITHUB-USERNAME.github.io/` after GitHub finishes deployment.

## Later custom domain

When `soviraglobal.com` is purchased, add it under Settings > Pages > Custom domain, then configure DNS at your domain provider.

## Important before launch

- Connect forms to Formspree, Netlify Forms, EmailJS, or your own backend.
- Replace placeholder product images with real photography when available.
- Add only verified certifications and documents.
